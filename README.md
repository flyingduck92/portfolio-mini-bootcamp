# 🌸 Cute-alism Portfolio

A high-performance, responsive portfolio website built with a **"Plan-First + RTCC-O"** workflow. This project blends the clean efficiency of **Minimalism** with the friendly, approachable aesthetics of **"Cute"** design (Soft UI).

## 🚀 Project Overview

This portfolio serves as a technical showcase and personal brand hub, designed to convert visitors through engaging micro-interactions and a high-end "Soft Tech" visual identity.

### 🛠 Tech Stack
- **Core:** HTML5, Vanilla JavaScript
- **Styling:** Tailwind CSS (Custom "Cute-alism" configuration)
- **Animation:** GSAP (ScrollTrigger, CustomEase)
- **Design Philosophy:** Cute-alism (Rounded corners, pastel accents, playful motion)

---

## 📂 Process Documentation

The development of this project followed a structured 4-step architectural process:

### [01. Brainstorming](./plan/01-brainstorm.md)
Focuses on the visual identity and interaction logic. 
- **Light/Dark Mode Strategy:** Soft Lavenders and Mints transitioning into Deep Charcoals and Sage.
- **Motion Logic:** "Squish" effects and elastic easing to reinforce the friendly brand.

### [02. Project Details (RTCC-O)](./plan/02-details.md)
Detailed breakdown of Roles, Tasks, Context, Constraints, and Objectives.
- **Constraint:** Zero-framework (Vanilla JS) to ensure maximum performance and lightweight footprint.
- **Objective:** Demonstrate technical mastery in frontend engineering and UI motion.

### [03. Execution Plan](./plan/03-execution.md)
A 4-phase implementation roadmap:
1. **Foundation:** Layout and Tailwind theme extension.
2. **Components:** Synthesis of complex UI elements (Spotlight cards, Shimmer accordions).
3. **GSAP Integration:** Sequencing the entrance animations and scroll-driven effects.
4. **Refinement:** Dark mode persistence and responsive polishing.

### [04. Results & Reflection](./plan/04-result.md)
Final evaluation of the build.
- **Outcome:** Sub-1s load times and seamless cross-device responsiveness.
- **Key Learning:** The synergy between "Corporate" structure and "Creative" visuals.

## ✨ Key Features & Components
Inspired by modern UI patterns and creative implementations:
- **Hero Section:** GSAP-staggered typography.
- **Vault (Projects):** Spotlight hover-effect cards with crossfade transitions.
- **Navigation:** Adaptive pill-shaped floating menu.
- **Interactive UI:** Shimmer accordions and magnetic CTA buttons.
